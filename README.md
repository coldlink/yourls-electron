# YOURLS Electron
Desktop menubar/taskbar app for the YOURLS url shortner made using electron

### Version
0.1.0

### Installation

Download latest packaged version of YOURLS Electron from https://github.com/coldlink/yourls-electron/releases. Windows and OSX.

### Dev Installation
```sh
$ npm install
$ npm start
```
